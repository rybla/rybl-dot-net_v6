I'm Henry Blanchette, and this is my personal website. Currently, I'm a PhD student at the University of Maryland. Read about my background in more detail (research, education and employment). I'm a researcher and programmer focused on:

  • developer tools for proof development and software engineering using type theory
  • programming language theory
  • formal specification and verification
  • user interface design
  • mathematical modeling and simulation

I am also especially interested in:

  • developer tools for proof development and software engineering using ML models
  • game design
  • economics and game theory

See my library of public writings on various topics usually related to my interests.
